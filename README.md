# Assessment 1 - Playwright + TypeScript

Overview:
This project is to fulfill an assessment for end-to-end purchase flow on the Swag Labs demo website using Playwright with TypeScript.

Website: [https://www.saucedemo.com](https://www.saucedemo.com)

Need to have:
- Node.js installed 
- Git installed 
- Playwright installed in project (`npm install`)

Steps needed to execute the little project of mine:

## 1. Clone the repository:

- git clone https://github.com/YOUR-USERNAME/Playwright-Arif.git
- cd Playwright-Arif

## 2. Install dependencies:
- npm install

## 3. Install Playwright browsers:
- npx playwright install

## 5. Running the test:
- npx playwright test
- but for beginner like me, i like to use -> npx playwright test --headed as you can view the ui hehe

## 6. View HTML Reporting:
- npx playwright show-report
